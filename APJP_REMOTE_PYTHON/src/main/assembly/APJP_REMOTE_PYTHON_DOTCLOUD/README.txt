PRE-INSTALLATION
----------------
- create DOTCLOUD account (http://www.dotcloud.com/)
- install DOTCLOUD CLI (http://docs.dotcloud.com/firststeps/)

INSTALLATION
------------
- open wsgi.py
	- configure
		- APJP.HTTP.APJP_KEY == generated key
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[0]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[0]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[1]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[1]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[2]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[2]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[3]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[3]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_KEY[4]
		- APJP.HTTP.APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_VALUE[4]
		- APJP.HTTPS.APJP_KEY == generated key
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[0]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[0]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[1]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[1]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[2]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[2]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[3]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[3]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_KEY[4]
		- APJP.HTTPS.APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_VALUE[4]
- install APJP FOR DOTCLOUD
	- open CONSOLE
		- dotcloud create application1
		- dotcloud push application1

IMPORTANT
---------
- APJP_REMOTE_HTTP_SERVER_REQUEST_URL == http://application-id.dotcloud.com/HTTP
- APJP_REMOTE_HTTPS_SERVER_REQUEST_URL == http://application-id.dotcloud.com/HTTPS
